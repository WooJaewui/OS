 # 파일과 디렉토리를 여러 사용들이 공동으로 관리하는 방법.

그룹.
그룹을 만들어서 파일을 생성한 유저 이외에 여러 사용자들이 공동으로 파일을 관리할 수 있다.



 # groupadd.

현재 디렉토리 확인.
ls -al에서 . 파일에 상세내용을 확인.

유저를 생성할 때 그룹 추가.
useradd -G 그룹이름 유저이름 

그룹 생성.
sudo groupadd 그룹이름
일반사용자는 group을 생성할 수 없다.

명령어 활용.
!! : 직전에 실행한 명령어를 의미한다.

그룹 생성 확인.
nano /etc/group에서 생성한 그룹이름 확인.

유저에 그룹 추가하기.
sudo usermod -a -G 그룹이름 유저이름
-a append : 그룹을 추가한다.
-G group : 그룹

파일 소유자 변경하기.
sudo chown ower : group 파일이름
