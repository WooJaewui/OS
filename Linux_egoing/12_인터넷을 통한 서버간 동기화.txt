 # rsync 1.

sync의 의미.
A기기와 B기기를 연결해서 한쪽에 정보의 변경이 일어났을 때 다른쪽 기기에 반영된다.

rsync.
remote sync는 원격으로( 인터넷을 통해 ) 컴퓨터에 정보를 다른 기기에 백업 등을 할 수 있는 방법이다.

rsync 명령어.
rsync -a 소스디렉토리/ 복사디렉토리 : 소스디렉토리 하위 파일들을 복사디렉토리에 복사한다.
rsync -av 소스디렉토리/ 복사디렉토리 : 소스디렉토리 하위 파일들 중에 복사디렉토리에 없는 것만 복사한다.



 # Remote sync

ip를 활용한 rsync.
rsync -azP 소스디렉토리/ ip주소:경로 : 소스디렉토리 하위 파일들을 ip주소 경로로 보내준다.
-z : 압축하고 전송한다.
-P : 전송되는 상황을 프로세스바로 보여준다.
