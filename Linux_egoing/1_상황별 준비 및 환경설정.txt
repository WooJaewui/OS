 # 맥 OS.

맥 OS.
맥은 Unix라는 OS를 사용하기 때문에 Linux를 따로 설치하지 않아도 된다.
맥을 CLI방식으로 다루는 방식을 배운다.

GUI, CLI 방식.
GUI : 컴퓨터를 그래픽( 이미지 )를 통해 쉽게 컴퓨터를 제어하는 방법.
CLI : 컴퓨터를 명령어를 통해 제어하는 방법. ( 터미널 사용 )
