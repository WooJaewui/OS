 # Linux 파일 경로와 순회.

파일 탐색 명령어들.
pwd : 현재 디렉토리 경로를 출력.
ls : 디렉토리 목록 나열. ( ls -l : 자세히보기 )
cd : 디렉토리를 변경.
clear : 화면 지우기.
man : 도움말 ( man ls, man cd, ... )

경로표시.
~ : 현재 로그인한 유저의 Home 디렉토리를 나타냄. ( /home/user이름 )



 # 파일 관리 명령어.

파일 관리 명령어.
mkdir : 디렉토리 생성.
rmdir : 디렉토리 삭제. 
touch : 빈 파일 생성.
mv : 파일 이동 / 변경.
rm : 파일 삭제. ( rm -r : 디렉토리 하위 파일들을 모두 지운다, -ri : 물어보면서 지움 )
cp : 파일 복사.

파일과 디렉토리.
파일이 디렉토리보다 큰 개념이다.
디렉토리도 파일의 한 종류이다. ( 디렉토리파일 )
ls -l에서 d~로 시작되는 파일을 디렉토리파일이다.
일반파일은 d가 없고 -가 있다.



 # 파일 편집 및 관리.

편집기.
VI : 모든 유닉스 편집기. ( 얕게 알아봄 )
GNU nano : 많이 사용되는 편집기. ( 조금 깊게 알아봄 )
보통 통합개발환경을 통해 개발을 하기 때문에 이런것이 있다라고 배움.

vi 편집기.
"vi 파일이름"을 통해 편집기 실행.
i : 내용을 입력.
o : 새로운 줄에서 입력.
a : 기존 입력 뒤에 입력.
ESC : 입력 끝내기.
:w : 저장하기.
:q : 나가기.
이동 : hl, kj
yy : 한 줄 복사.
yw : 한 단어 복사.
yl : 한 문자 복사.

나노 실행.
nano 파일이름.

파일 실행.
tap : 현재 디렉토리 파일중에 자동완성을 해준다. ( 두 개 이상일경우 목록을 보여줌 )

나노 팁.
아래 부분에 모든 설명이 나와있다.

 # 파일 찾기와 파일 정보 확인.

파일 찾기 명령어.
find ./ -name *.java

찾기 명령어.
-name : 이름으로 찾는다.
-size : 크기로 찾는다.
권한에 따라 못 찾을수도 있다.

파일 크기.
b : 기본설정 512
c : byte.
w : 2byte.
k : Kilobyte.
M : Megabyte.

파일 정보 확인하기.
cat 파일이름 : 파일 내용 보기.
head -n2 파일이름 : 위에서부터 2줄만 보기.
tail -n2 파일이름 : 아래서부터 2줄만 보기.
grep 찾고싶은단어 파일이름 : 파일에서 찾고싶은단어가 있는 부분을 보기. ( 띄어쓰기 "단어"로 표시 )
grep -i : 소대문자 구분없이 찾기 옵션.

ls 활용하기.
ls *단어 : 디렉토리에서 아무글자 + 단어인 모든 파일 보기.
ls ?단어 : 디렉토리에서 한 글자 + 단어인 모든 파일 보기.
ls [12]단어 : 디렉토리에서 1 또는 2 + 단어인 모든 파일 보기.

두 파일 비교하기.
cmp 파일이름1 파일이름2 : 다른부분이 몇번째 줄인지 알려준다.
diff 파일이름1 파일이름2 : 다른부분을 직접 보여준다.

파일 정보보기.
file 파일이름 : 파일에 대한 정보를 보여준다.
